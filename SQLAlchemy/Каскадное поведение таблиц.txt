CASCADE

save-update - при добавлении объекта в сессию, объекты связанные с ним через relationship(), также попадают в сессию

dalete - при удалении родительского объекта будут удалены все связанные с ним дети

delete-orphan - при удалении ребенка из родительского объекта, произойдет удаление элемента из таблицы детей


LAZY

ленивая загрузка (lazy, по умолчанию) - при запросе объекты загружаются без связанных таблиц

жадная загрузка (нетерпеливая, eager) - при запросе объекты загружаются вместе со связанными объектами

отсутствие загрузки (no loading) - отключение загрузки для связанных таблиц
